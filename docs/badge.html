Badge

Badge can be used to show the status information of the user or some count information such as notification numbers or number of items in the cart.

Badge on Avatar

Badge on Avatar can show different user status, such as offline and online. We have different badges that can be used with avatar.

Badge status includes online badge, offline badge, and number badge. You can add this badges at four positions, top left, top right, bottom right, and bottom left.

Badge on Icons

Number badges on icons is commonly used pattern to display number counts of notification or any other countable thing.
